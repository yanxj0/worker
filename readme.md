# Web Worker使用

## 内容
表达的内容为，当处理较大数据量时，使用webworker和不使用时浏览器窗口是否表现出卡顿。   

数据量有3W条，为表现明显一点，首先是将嵌套的数据转为一维数组，然后在将一维数组转为嵌套数据，在用ztree渲染
